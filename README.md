# spanish-vocabulary

一个开源西班牙语词库仓库，收录并整理了《现代西班牙语学生用书》系列词汇。

这个仓库就是 [`A4-Memory`](https://github.com/k7tmiz/A4-Memory) 的西班牙语线上导入词书仓库，用来托管可被 `A4-Memory` 直接读取的远程 JSON 词书。词库采用简洁统一的结构，方便线上导入、维护和后续扩展。

## 项目定位

- 作为 `A4-Memory` 的西班牙语线上词书仓库
- 提供可直接读取的西班牙语 JSON 词书
- 保持统一、简单、稳定的数据格式
- 方便后续继续补充更多教材、分级词表或主题词库

## 数据格式

每个词书文件都是一个独立 JSON，对应一个完整词书或一课词汇。

顶层结构：

```json
{
  "name": "现代西班牙语学生用书1",
  "description": "现代西班牙语学生用书1",
  "language": "es",
  "words": [
    {
      "term": "cómo",
      "pos": "adv.",
      "meaning": "怎么"
    }
  ]
}
```

字段说明：

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `name` | `string` | 词书名称 |
| `description` | `string` | 词书描述 |
| `language` | `string` | 语言代码，目前为 `es` |
| `words` | `array` | 词条数组 |
| `term` | `string` | 西班牙语词汇或固定搭配 |
| `pos` | `string` | 词性，可为空字符串 |
| `meaning` | `string` | 中文释义 |

设计原则：

- 结构尽量扁平，便于不同工具复用
- 单词、短语、固定搭配统一使用 `term`
- 不强制附加复杂元数据，降低导入门槛
- 保留教材词条原有表达方式，优先保证可学可用

## 与 A4-Memory 的关系

[`A4-Memory`](https://github.com/k7tmiz/A4-Memory) 是一个纯前端背单词工具，支持自定义词书导入、在线词书导入、打印、复习与发音。

本仓库不是泛用示例词库，而是 `A4-Memory` 已经接入的西班牙语线上词书源。这里的 JSON 文件会作为远程词书被读取，用于把教材词汇直接接入记忆流程。

两边的数据结构天然接近：

- `language` 对应词书语言
- `words` 是词条数组
- 词条核心字段使用 `term`、`pos`、`meaning`

这意味着你可以把本仓库看作：

- `A4-Memory` 的西班牙语线上词书后端数据仓库
- 教材词汇到记忆系统之间的数据中间层
- 后续扩展更多西语词书时的统一数据基座

## 在 A4-Memory 中使用

`A4-Memory` 已经对接本仓库，所以这里的核心用途不是“手动下载后再试着导入”，而是作为线上词书源持续维护。

### 1. 线上导入

`A4-Memory` 可以直接读取本仓库中的远程 JSON 文件，把它们作为西班牙语线上词书。

当前主要入口就是这些 Raw 地址：

```text
https://raw.githubusercontent.com/k7tmiz/spanish-vocabulary/main/现代西班牙语学生用书1.json
https://raw.githubusercontent.com/k7tmiz/spanish-vocabulary/main/现代西班牙语学生用书2.json
https://raw.githubusercontent.com/k7tmiz/spanish-vocabulary/main/现代西班牙语学生用书3.json
```

适合：

- 在 `A4-Memory` 中直接作为线上词书使用
- 统一维护远程词书内容
- 更新后由 `A4-Memory` 同步读取最新版本

### 2. 本地导入或二次处理

如果你也想单独使用这些词书，依然可以把 JSON 下载到本地，导入其他工具，或者作为自己的词库原始数据继续处理。

适合：

- 本地学习
- 手动裁剪某一课或某一册
- 转换成 Anki、CSV 或其他格式

如果需要为 `A4-Memory` 额外补充 `id`、标签、分组信息，建议在消费层做轻量转换，尽量不要破坏这里作为原始线上词书仓库的简洁结构。
